  1. Links and Notes for Week 1 of RampUp

    Email from Alex Boerstler

    Hey guys!

    Hope you've had a good weekend and a moment to breathe to let last Thursday's introduction to front-end settle in. Just to review...

    We went over what Git is, and what github is, the difference between the two and how to use the github app to do version control. If you'd like to dive deeper into git, github, and version control, try doing the tutorials github shows you when you log in. You can also access those articles here: https://help.github.com/categories/54/articles If you're interested in trying the command line without having to set everything up, Github has an online tutorial here: https://try.github.io/levels/1/challenges/1

    Next, we went over what an IDE is, and how to set up Sublime Text, a programming text editor that's an extensible IDE. We installed Package Control, and you guys might want to take a look at this guide by Drew Bartolini on how to configure sublime. Also, my favorite theme is Spacegray, but you should do some exploration and find something you like.

    After that, we started writing some HTML. Mozilla's fantastic developer network has a list of HTML5 semantic tags you guys should look at. There's also this super minimalist reference guide for both tags and attributes you can read (click the element or attribute you want data on).

    After that we dove into CSS. For a good overview and extensive documentation, Mozilla again helps us out with its guide to CSS selectors. If you'd like to dive further into css, The Magic of CSS is a good guide and CSS-Tricks is one of the general CSS resources I always turn to.

    Note that some of this stuff is beyond what we covered; have fun exploring, don't worry if you don't understand it all.

    Looking forward to seeing you guys on Thursday!
